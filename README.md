# LiveOU V4 Online Authorization Registry

This public repository contains only a signed authorization-status registry for LiveOU V4 desktop clients.

- It contains no application source code.
- It contains no private signing key, customer activation code, token, cookie, or personal data.
- Client applications verify the signature before trusting the registry.
- A listed license can be enabled or revoked by the software issuer.

Do not manually edit `authorization_registry.json`; update it only with the issuer's offline signing tool.
